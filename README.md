#### mkworld   

---
自建 `planet` 时需要生成行星文件，这是在 [ZerotierOne](https://github.com/zerotier/ZeroTierOne) 官方库整理出来的代码，传入 `moon.json` 文件生成，该生成代码参考了[kaaass](https://github.com/kaaass/ZeroTierOne/blob/master/attic/world/mkmoonworld.cpp) 的代码   
---  
使用 `xmake` 直接编译即可
```bash
xmake
```
